## Instrucciones de Implementación

1. **Generar cada imagen** usando Midjourney / DALL-E 3 / Leonardo.ai con el prompt proporcionado
2. **Exportar en PNG** con fondo transparente y al menos 1024×1024 px
3. **Optimizar tamaño** (sin pérdida visible de calidad, apuntar a < 500 KB por imagen)
4. **Guardar con el nombre exacto** indicado en `public/stack/<tecnologia>/`
5. **Notificar** cuando estén listas para que las integre automáticamente en cada página
