# Roadmap

## ✅ Hecho — Fundación y experiencia

- Proyecto Next.js 16 + React 19 + TS + Tailwind v4.
- Sistema de diseño (tokens dark-first + tema claro, glass, glow, gradientes).
- Layout: header glass con scrollspy, footer, tema claro/oscuro, smooth scroll (Lenis).
- **Hero 3D** (React Three Fiber): núcleo de IA, geometrías, partículas, bloom, parallax; typewriter, CTAs, stats.
- Secciones: Sobre mí, Stack (tilt cards + niveles), Proyectos, Experiencia (timeline animado), Contacto (RHF + Zod).
- **Case studies** por proyecto (`/projects/[slug]`) + índice `/projects`.
- **Blog MDX** (`/blog`) con 3 artículos y JSON-LD (Article/FAQ/Breadcrumb).
- SEO: sitemap, robots, manifest, OG dinámica, datos estructurados.
- Accesibilidad WCAG 2.2 y `prefers-reduced-motion` en todo el movimiento.

## 🔜 Personalización (Jorge)

- Subir CV en `public/resume/`.
- Confirmar redes y dominio en `content/site.ts`.
- Foto profesional en "Sobre mí".
- Capturas/vídeos reales de proyectos en `public/projects/`.
