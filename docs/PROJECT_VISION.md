# Visión del Proyecto

## El objetivo

Construir el mejor portafolio que exista para un Automation Engineer / AI Engineer / RPA Developer. **No un portafolio tradicional**, sino algo que parezca la página principal de una empresa tecnológica de Silicon Valley.

Efecto WOW desde el primer segundo. Que cualquier reclutador piense: *"Necesitamos entrevistar a esta persona inmediatamente."*

## Principios

1. **Cada sección debe sorprender.** Nada estático; todo se siente vivo con propósito.
2. **Cada animación tiene una razón.** El movimiento comunica ingeniería, no decora.
3. **Rendimiento como feature.** Core Web Vitals > 95. La belleza nunca justifica un sitio lento.
4. **Accesible por defecto.** WCAG 2.2 AA. La excelencia incluye a todos.
5. **Marca sobre currículum.** No listamos tareas; contamos una historia de impacto.

## La experiencia, sección por sección

- **Hero 3D** — un núcleo de IA que respira, geometrías flotantes, partículas y parallax con el mouse. Titular con escritura dinámica.
- **Sobre mí** — narrativa de marca con foco en valor, no en tareas.
- **Stack** — dominio técnico como una tabla periódica interactiva de habilidades.
- **Proyectos** — casos de éxito, no tarjetas: problema → solución → arquitectura → resultados.
- **Experiencia** — timeline que se dibuja con el scroll.
- **Blog** — autoridad técnica optimizada para SEO, AEO y GEO.
- **Contacto** — conversión con validación real y microinteracciones.

## Criterio de éxito

El resultado final debe ser digno de competir por **Awwwards, CSS Design Awards y FWA**, y —más importante— convertir visitas en entrevistas.
